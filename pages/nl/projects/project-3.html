Waarom heb ik dit project gemaakt?

Ik heb een schoolproject waarbij we onze professioneel ogende website moeten maken zonder een websitebouwer zoals wordpress of wix enz.

Wat heb ik geleerd?

Ik heb het belang geleerd van het ontwerpen van een website, een van de dingen die mijn leraar zei en echt logisch is: "Voordat je een site wilt maken, is het belangrijk om het eerst te ontwerpen, want als je dat niet doet, kan het echt verwarrend voor je zijn". Dat liet me denken, verdomd, hij heeft gelijk, ik moet het ontwerpen! Maar dat beviel me in eerste instantie niet, maar uiteindelijk was het de moeite waard! Ik kreeg inzicht in html, css & js en hoe je deze drie programmeertalen kunt combineren om iets speciaals te creëren!
